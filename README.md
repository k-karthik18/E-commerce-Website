# E-Commerce Website

A fully responsive, multi-page e-commerce front-end template built entirely with core web technologies. This project provides a complete structural flow for an online store, from browsing a product catalog to reviewing items in the shopping cart.

## Features

* **Complete Storefront Flow:** Includes seamlessly connected pages for Home, Shop, Product Details, Blog, About, Contact, and Cart.
* **Product Catalog:** Features a primary storefront (`shop.html`) and an individual product view (`sproduct.html`) to display item details.
* **Shopping Cart Interface:** A dedicated cart page (`cart.html`) to review selected items before checkout.
* **Responsive Styling:** Custom, framework-free CSS (`style.css`) designed to adapt across different screen sizes and devices.

## Tech Stack

* **HTML5:** Semantic markup for all site pages.
* **CSS3:** Custom styling and layout management.

## Project Structure

* `index.html` - The main landing/home page.
* `shop.html` - The primary product catalog.
* `sproduct.html` - Detailed view for individual products.
* `blog.html` - Section for articles and store updates.
* `about.html` - Information about the brand/store.
* `contact.html` - Contact form and location details.
* `cart.html` - The shopping cart interface.
* `style.css` - Global stylesheet managing the site's design.
* `/img` - Directory containing all image assets used across the site.
